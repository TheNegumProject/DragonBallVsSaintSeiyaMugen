Marvel vs Capcom-Eternity of Heroes Gouki by vyn
---------------------



  Controls
  --------

    Basic Controls
    --------------
      

      We will use the notation:

      D - down
      F - forward
      U - up
      B - back

      x - weak punch
      y - medium punch
      z - strong punch
      a - weak kick
      b - medium kick
      c - strong kick

 

    Standard Movement
    -----------------
      U              - Jump
      UF             - Jump forwards
      F              - Walk forwards
      DF             - Crouch
      D              - Crouch
      DB             - Crouch or low guard
      B              - Walk backwards or high guard
      UB             - Jump back
      F, F           - Forward dash
      B, B           - Back dash
      xy             - Recovery (when falling)

--------Throws---------

- a + x (when close) - Note: Hold back while grabbing to throw back

--------Technical Abilities---------

-Guard Push-

 PP (while guarding)

-Ground Recovery-

 F, F while lying down

-Zero Counter-

  D, DB, B, K (while guarding)

  D, DB, B, P (while guarding)

--------Ground Specials---------

-Gou Hadou-

  D, DF, F, P

-Fire Gou Hadou-

  B, BD, D, DF, F, P 

-Gou Shoryuu-

  F, D, DF, P

-Gou Senpuu-

  D, DB, B, K

--------Air Specials---------

-Air -Gou Hadou--

  D, DF, F, P

-Falling kick-

  D, DF, F, k

NOTES: Medium Kick version causes gouki to bounce when it hits, 
       repeat the command to execute the move again.

-Air Gou Senpuu-

  D, DB, B, K

-Air Float Gou Senpuu-

  F, FD, D, DB, B, K

--------Ground Supers---------

-Messastu Gou Hadou-

  D, DB, B, PP

-Messastu Gou Shoryuu-

  D, DF, F, PP

-Messastu Gou Tatzumaki-

  D, DB, B, KK

-Misogi- (Requires 2 super bars)

  D, D, KK

-Shun Goku Satsu- (Requires 3 super bars)

  x, x, F, a, z


--------Air Supers---------

-Tenma Gou Zankuu-

  D, DF, F, PP

-Messastu Gou Tatzumaki-

  D, DB, B, KK
